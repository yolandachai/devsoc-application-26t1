# devsoc-application-26t1
Website based on the theme "Day and Night". Created for the DevSoc Training Program 26T1 Trainee Application.

Please find the deployed page at this link: https://yolandachai.github.io/devsoc-application-26t1

### Day and Night Vampire Dress-Up
Modelled after the flash dress-up games that were popular in the 2010s. Create two alternate outfits, inspired by what a vampire would wear in the daytime and nighttime!
